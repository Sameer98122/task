Frequently Asked Questions

Why no JavaScript?

Modern CSS can handle many interactive patterns once reserved for JavaScript, reducing bundle size and improving accessibility. This showcase demonstrates checkboxes, radio buttons, :target, flip transforms, and peer selectors in action.

Is this mobile-friendly?

Yes. Tailwind’s responsive classes ensure the layout adapts from small screens (single-column) to wide desktops (three-column grids).

Can I print my herbarium?

Absolutely. The markup is semantic and print-friendly; you may add a @media print layer in Tailwind to adjust colors.
